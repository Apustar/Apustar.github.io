#Ubuntu server16.04下配置JAVA及TOMCAT+Apache-Maven

###在Ubuntu server16.04LTS中配置Java及Tomcat，一堆坑啊。

###1.首先建立Java文件夹

###在shell下执行

sudo mkdir /usr/local/Java

###2.给usr目录设置读写权限，否则会出现Permission denied错误，导致无法下载java及tomcat。

sudo chmod -R 777 /usr/local/Java

###3.进入刚才建立的目录

cd /usr/local/Java

###4.下载最新的JAVA，其中http后的下载路径需要自己到Oracle网站获取

sudo wget --no-check-certificate --no-cookies --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/jdk-8u131-linux-x64.tar.gz

###5.解压java

tar -zxvf jdk-8u131-linux-x64.tar.gz

###6.设置JAVA环境变量

sudo vim ~/.bashrc
//点击E进行编辑
//点击i插入
//在文件的最后添加
#set oracle jdk environment
export JAVA_HOME=/usr/local/Java/jdk1.8.0_131  # 这里的目录要换成你解压的JDK 目录
export JRE_HOME=${JAVA_HOME}/jre  
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib  
export PATH=${JAVA_HOME}/bin:$PATH  

//点击ESC退出插入
//点击ZZ保存退出

###7.使环境变量马上生效

source ~/.bashrc

###8.检查Java是否安装成功

java -version

###这里写图片描述

###看到版本信息为1.8.0_131，java安装及配置成功。

###9.下载Tomcat，其中Tomcat下载地址需自行到tomcat.apache.org网站获取

wget http://mirror.bit.edu.cn/apache/tomcat/tomcat-8/v8.5.16/bin/apache-tomcat-8.5.16.tar.gz

###10.解压tomcat

tar -zxvf apache-tomcat-8.5.16.tar.gz

###11.进入tomcat安装目录下的bin目录

cd /usr/local/Java/apache-tomcat-8.5.16/bin

###12.编辑startup.sh，添加环境变量

vi startup.sh
//在这个文件的最后一行上面添加以下配置，编辑方法同上：
JAVA_HOME=/usr/local/Java/jdk1.8.0_131
JRE_HOME=${JAVA_HOME}/jre
PATH=${JAVA_HOME}/bin:$PATH
CLASSPATH=.:${JAVA_HOME}/lib/dt.jar:${JAVA_HOME}/lib/tools.jar
TOMCAT_HOME=/usr/local/Java/apache-tomcat-8.5.16

chmod +x startup.sh
chmod +x shutdown.sh

###15.启动Tomcat

./startup.sh

###ubuntu16.04安装maven

###maven是个项目管理工具，在编程领域应用广泛。本文主要讲述如何在ubuntu16.04系统下安装maven。

###第一步，去官网下载maven。

###这里写图片描述

###第二步，解压到/opt/maven目录。

###创建manve目录。

sudo mkdir /opt/maven

###解压到/opt/maven目录下。

sudo tar zxvf apache-maven-3.5.0-bin.tar.gz -C /opt/maven

###第三步，配置mavne环境。

sudo gedit /etc/profile

###在文件内容后面添加以下内容：

export M2_HOME=/opt/maven/apache-maven-3.5.0
export CLASSPATH=$CLASSPATH:$M2_HOME/lib
export PATH=$PATH:$M2_HOME/bin

保存，输入以下命令使配置文件生效。

source /etc/profile

验证是否安装成功。

mvn -v
